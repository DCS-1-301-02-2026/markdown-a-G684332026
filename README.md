# 情報工学科の紹介
G684332026 大原　瑞希
## 情報工学とは
情報工学とは、**コンピュータ技術**、**ネットワーク技術**を活用して、情報を効率よく処理・活用するための学問です。
###　学べる内容
- プログラミング
- ネットワーク
- データベース
- AI(人工知能)
- セキュリティー
### 情報工学科の特徴
1. 実践的なプログラミング学習
2. 最新技術を学べる
3. 就職に強い💪
### 将来の仕事
- システムエンジニア
- webエンジニア
- AI開発者
- ゲームプログラマー
